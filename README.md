# Java 2 Player Headball Football ⚽🎮

A simple 2D Java game built using the CityEngine physics engine.

## Gameplay
A two-player game where both players move around the world and collect stars ⭐. The player with the most stars wins (no win screen yet).

## Controls

**Player 1**
- A / D → Move left/right  
- W → Jump  

**Player 2**
- ← / → → Move left/right  
- ↑ → Jump  

## How to Run
1. Open the project in IntelliJ IDEA  
2. Go to File → Project Structure → Libraries  
3. Click + → Java and select lib/CityEngine.jar  
4. Apply changes  
5. Run Game.java  

## Project Structure
- src/ → game code  
- data/ → images/assets  
- lib/ → required libraries  

## Notes
- Make sure the data folder stays in the project root or images won’t load
